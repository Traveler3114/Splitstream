# Image Assets

This directory contains image assets used in the README documentation.

## Required Images

Add the following images to this directory:

### `banner.png`
- Main banner image for the README header
- Recommended size: 1200x400px or similar 3:1 aspect ratio
- Should showcase the game's key features and atmosphere

### `video-thumbnail.png`
- Thumbnail for the setup tutorial video
- Recommended size: 1280x720px (16:9 ratio)
- Include a play button overlay to indicate it's a video link

## Optional Images

Consider adding these additional images to enhance documentation:

- `gameplay-screenshot-1.png` - Stealth gameplay example
- `gameplay-screenshot-2.png` - Timeline switching demonstration
- `gameplay-screenshot-3.png` - Puzzle solving example
- `ui-screenshot.png` - User interface showcase
- `multiplayer-screenshot.png` - Co-op gameplay
- `level-design.png` - Environment showcase

## Image Guidelines

- **Format**: PNG for screenshots, JPG for photos
- **Quality**: High resolution but optimized for web (< 1MB per image)
- **Naming**: Use descriptive kebab-case names (e.g., `stealth-gameplay.png`)
- **Attribution**: Include credits if using third-party images

## Adding Images

1. Place image files in this directory
2. Reference them in README.md using relative paths:
   ```markdown
   ![Alt Text](docs/images/filename.png)
   ```
3. Commit and push the images to the repository
